# PC Dotfiles

This repository contains my personal dotfiles for my PC. It includes configuration files for various programs and scripts to automate the installation of these files.

This is targeted at a NixOS system!

## Aquiring the dotfiles

To aquire the installation script(s) for these dotfiles simply clone the repository to your home directory.

```bash
git clone https://github.com/soriphoono/dotfiles.git ~/.dotfiles
```

## Installation

To install these dotfiles simply run the installation script, available for viewing [here](install.sh).

```bash
cd && sh .dotfiles/install.sh
```
